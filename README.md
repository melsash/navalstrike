# ⚓ NavalStrike — Морской бой нового поколения

> Современная мультиплеер-платформа для игры в Морской бой с AI-тренером, глобальным рейтингом и умным ботом.

🎮 **Live Demo:** [navalstrike.vercel.app](https://navalstrike.vercel.app)

---

## 🚀 Что это?

NavalStrike — это не просто «ещё один морской бой». Онлайн multiplayer Battleship с AI-анализом партий и realtime синхронизацией.
**Для кого:** люди которые хотят сыграть в классику с другом онлайн, не скачивая ничего.  
**Почему это ценно:** мгновенный мультиплеер по ссылке + AI-разбор после каждой партии = возвращаемость пользователей.

---

## ✨ Фичи

### 🎮 Игровой процесс
- Полные правила Морского боя (расстановка, очерёдность, потопление)
- Расстановка кораблей вручную с превью или автоматически
- Умный бот с алгоритмом Hunt & Target (охота по шахматному паттерну, добивание подбитых)

### 🌐 Мультиплеер
- Игра с другом по 6-значному коду комнаты
- Realtime синхронизация через Supabase Realtime (WebSockets)
- Никакой регистрации для входа в чужую комнату не нужно — просто код

### 🤖 AI Coach
-После каждой партии AI анализирует твою стратегию
- Конкретные советы: «ты стрелял хаотично» или «хорошо добивал корабли»
- Персональный разбор на основе реальной статистики партии

### 🏆 Глобальный рейтинг
- Таблица лидеров по победам
- Точность стрельбы с прогресс-баром
- Подсветка твоей позиции
- Медали 🥇🥈🥉 для топ-3

### 👤 Профиль и статистика
- Авторизация через email
- Личная статистика: победы, поражения, точность
- PRO-статус (готова монетизация)

---
## 🧠 Архитектура

- Frontend построен на Next.js App Router с клиентскими React-компонентами
- Игровая логика (бот, проверка попаданий, победа/поражение) реализована на TypeScript
- Multiplayer работает через Supabase Realtime WebSockets
- Данные игроков, комнат и статистики хранятся в PostgreSQL (Supabase)
- AI Coach вызывается через API route после завершения матча
- Deploy и CI/CD выполняются через Vercel + GitHub integration

### Поток игры

```text
Игрок → Next.js UI → API Routes → Supabase DB/Realtime
                                ↓
                          AI Coach API
```

## 💡 Продуктовое мышление

### Почему NavalStrike выделяется:

**Проблема:** существующие сайты для морского боя — это либо скучные одиночные игры, либо перегруженные платформы требующие регистрации обоих игроков.

**Решение:** мгновенный мультиплеер по ссылке (как в Skribbl.io) + AI-тренер для удержания (как в Chess.com).

### Монетизация (готова к реализации):
- **PRO аккаунт** — кастомные скины кораблей, расширенная AI-аналитика
- **Турниры** — платные турнирные сетки для школ и офисов
- **API** — белый лейбл для образовательных платформ

### Метрики удержания:
- AI Coach создаёт желание «сыграть ещё раз и улучшить результат»
- Лидерборд создаёт соревновательность
- Мультиплеер по ссылке = виральность через шаринг

## Запуск локально

```bash
git clone https://github.com/melsash/navalstrike.git
cd navalstrike
npm install

# Создай .env.local
cp .env.example .env.local
# Заполни переменные окружения

npm run dev
```

### Переменные окружения:
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
HUGGINGFACE_API_KEY=your_key
```
